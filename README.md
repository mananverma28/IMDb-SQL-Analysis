# IMDb Movies & TV Shows Analysis using SQL

## Project Overview

This project analyzes an IMDb Movies & TV Shows dataset using SQL to
uncover insights about movies and TV shows, including ratings, release
trends, genres, runtimes, certifications, and production countries.

## Objectives

-   Analyze Movies and TV Shows.
-   Identify highest-rated content.
-   Analyze release year trends.
-   Explore age certifications.
-   Identify top production countries.
-   Calculate average IMDb ratings and runtimes.

## SQL Skills Used

-   SELECT
-   WHERE
-   GROUP BY
-   ORDER BY
-   LIMIT
-   COUNT()
-   AVG()

## Business Questions

-   Total number of titles
-   Movies vs TV Shows
-   Highest-rated movies
-   Highest-rated TV shows
-   Most common age certification
-   Top production countries
-   Average IMDb score
-   Average runtime

## Dataset

IMDb Movies & TV Shows Dataset (Kaggle)
